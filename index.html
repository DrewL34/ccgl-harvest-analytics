Set up GitHub access

To Publish your edits, the dashboard needs a Personal Access Token with write access to the inventory repo. One-time setup, ~3 minutes.

  1. Go to github.com/settings/personal-access-tokens/new
  2. Token name: CCGL Inventory
  3. Repository access → Only select repositories → pick ccgl-inventory
  4. Repository permissions → Contents → Read and write
  5. Click Generate token. Copy the github_pat_... string.
  6. Paste it below:

Token is stored only in your browser's localStorage — it never leaves this device. Click "sign out" to clear it.
